# 个人助理 PWA

一个具有毛玻璃质感UI的渐进式Web应用，包含习惯打卡、训练记录、灵感捕捉和短视频收藏功能。

## 功能特性

### 1. 习惯打卡
- ✅ 自定义添加习惯（带图标选择）
- ✅ 每日点击打卡
- ✅ **打卡后弹出鼓励弹窗**（随机显示鼓励语）
- ✅ 连续打卡天数统计
- ✅ 今日完成/总习惯/连续天数统计

### 2. 训练记录
- ✅ 新建训练日志
- ✅ 8种常用动作快速选择
- ✅ **组间休息计时器**（顶部自动显示）
- ✅ **动作详解**（点击"详解"按钮查看）
- ✅ 记录重量、次数、组数
- ✅ 本周训练/总容量/总组数统计

### 3. 灵感捕捉
- ✅ 快速记录灵感
- ✅ **落地勾选框**（标记灵感已完成）
- ✅ 时间倒序浏览
- ✅ 完成状态显示（划线效果）

### 4. 短视频稍后再看
- ✅ 粘贴链接自动识别平台
- ✅ 自动归类标签（短视频/长视频/海外/生活）
- ✅ 点击卡片跳转播放
- ✅ 支持抖音、B站、YouTube、小红书、快手

## 设计特色

- **毛玻璃（Glassmorphism）UI**：半透明背景 + 模糊效果
- **深色渐变主题**：紫色到蓝色的优雅渐变
- **流畅动画**：页面切换、弹窗、悬停效果
- **底部导航**：类似原生APP的Tab切换

## 使用方法

### 方式一：直接打开（最简单）
1. 用浏览器打开 `index.html` 文件
2. 即可开始使用

### 方式二：添加到主屏（推荐）
1. 在手机上用浏览器打开
2. 点击浏览器菜单 → "添加到主屏幕"
3. 之后像原生APP一样从桌面打开

### 方式三：部署到服务器
将文件夹上传到任何静态托管服务：
- GitHub Pages
- Vercel
- Netlify
- 腾讯云/阿里云OSS

## 技术栈

- **纯前端**：HTML5 + CSS3 + JavaScript
- **PWA**：Service Worker + Manifest
- **存储**：LocalStorage（本地持久化）
- **UI**：CSS backdrop-filter 毛玻璃效果

## 文件结构

```
│   └── icon.svg
└── README.md
```

## 数据说明

所有数据保存在浏览器本地（LocalStorage），
- ✅ 离线可用
- ✅ 清除浏览器数据会丢失
- ❌ 无法跨设备同步

如需云同步，可后续接入 Supabase/Firebase。

## 浏览器兼容性

- ✅ Chrome/Edge（推荐）
- ✅ Safari（iOS）
- ✅ Firefox
- ✅ 微信内置浏览器

**注意**：毛玻璃效果在部分旧版浏览器可能不显示，但功能正常。
